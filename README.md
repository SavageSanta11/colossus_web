This is the repo containing the web interface for Colossus.

### Create an envrionment

#### macOS/Linux
```
$ python3 -m venv venv
```

#### Windows
```
> py -3 -m venv venv
```

### Activate the environment

#### macOS/Linux
```
$ . venv/bin/activate
```

#### Windows
```
> venv\Scripts\activate
```

### Install dependencies
```
$ pip install -r requirements.txt
```

### Run the app
```
$ flask --debug run
```
